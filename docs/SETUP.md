# Setup BOT-ARQ V4.2

1. Copiar el contenido del ZIP a la raíz del repositorio.
2. No borrar `.git`.
3. Commit y push con GitHub Desktop.
4. Ejecutar workflow manualmente con `force=true`.
5. Revisar `datos_acciones.json` y `paper/*.json`.

La configuración real está en:

`config/system_config.json`
