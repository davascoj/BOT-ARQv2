# Setup BOT-ARQ V4.1

1. Copiar todo el contenido del ZIP en la raíz del repositorio local.
2. No borrar `.git`.
3. Aceptar reemplazo de archivos existentes.
4. Borrar `__pycache__` si aparece.
5. Commit en GitHub Desktop.
6. Push origin.
7. Ejecutar GitHub Actions manualmente con `force=true`.
8. Refrescar GitHub Pages con Ctrl + F5.
